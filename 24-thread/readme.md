
# Threading

## Apps included:


### video2audio:
This app will change the videos in mp4 to audios in mp3.

#### Usage
Just add your videos into video folder.

Install requiremnets:
```
pip install -r requiremnts.txt
```
Then run:
```
python main.py

```

The exit will be in the audio folder.
